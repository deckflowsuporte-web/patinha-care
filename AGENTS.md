# 🎯 Creative Studio

Estúdio criativo com **Three.js**, **Remotion** e **animações avançadas**.

## 📁 Estrutura

```
/workspace/project/
├── .config/              # ⚙️ Tudo aqui dentro
│   ├── src/             # Código fonte
│   ├── .agents/         # Skills (ver .agents/skills/SKILLS.md)
│   ├── package.json     # Dependências
│   └── ...
└── .git/                # Git
```

## 🚀 Comandos

```bash
npm run install:deps   # Instalar dependências
npm run dev            # Development
npm run build          # Build
```

## 📖 Docs

- `.config/README.md` - Documentação completa
- `.config/src/lib/skills-manager.ts` - Gerenciador de skills
- `.config/.agents/skills/SKILLS.md` - Índice de skills
- `.config/.agents/skills/` - Skills instaladas (threejs-*, remotion, gsap-*)
